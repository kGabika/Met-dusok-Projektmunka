Metódusokhoz kapcsolódó alapfogalmak

A metódusok a C# nyelvben olyan kódrészletek, amelyek egy adott feladatot hajtanak végre. Egy metódus felépítése:

  • Fejléc: Meghatározza a metódus nevét, paramétereit és visszatérési típusát
  • Törzs: A metódus végrehajtandó utasításait tartalmazza
  • Visszatérési érték: A metódus eredménye, ha van

Szintaxis példa:


// Egy egyszerű metódus C#-ban
public int Osszead(int a, int b)
{
    return a + b;
}

Metódus hívása:


int eredmeny = Osszead(3, 5);

Metódustípusok

Eljárás (void típusú metódus)

Az eljárás nem ad vissza értéket csak végrehajt egy műveletet


public void Udvozles()
{
    Console.WriteLine("Üdvözlöm!");
}

Függvény (visszatérési értékkel)

A függvény visszaad egy értéket


public double Atlag(double osszeg, int db)
{
    return osszeg / db;
}

Metódusok összehasonlítása

Típus Visszatérési érték Példa
Eljárás nincs public void Udvozles()
Függvény van public int Osszead(int a, int b)
